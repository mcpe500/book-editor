# SPEC: Book Editor - React + Vite + Express Architecture

**Version:** 2.0.0
**Created:** 2026-04-16
**Project:** Book Editor untuk Academic Thesis Writing
**Architecture:** React + Vite (Frontend) + Express.js (Backend) + JSON Files (Database)

---

## 1. Executive Summary

### 1.1 Background

Migrasi dari SvelteKit ke arsitektur terpisah frontend-backend. Alasan: ketidakpuasan dengan struktur SvelteKit. Keputusan: React + Vite untuk frontend, Express.js untuk backend, file-based JSON untuk database.

### 1.2 Architecture Decision

| Layer | Technology | Rationale |
|-------|------------|-----------|
| Frontend | React 19 + Vite 5 + TypeScript | Familiar, modern tooling, fast HMR |
| Backend | Express.js + TypeScript | Mature, stable, good ecosystem |
| Database | JSON Files | Simple, no setup, portable |
| File Storage | Local filesystem | Simple, no cloud dependency |
| Auth | None | Single-user app, MVP scope |

### 1.3 Key Features (Carried from v1)

1. **Book Management** - Create, edit, delete books
2. **Chapter Organization** - Hierarchical chapter/section structure
3. **Rich Content Editing** - Markdown + LaTeX + Images + Tables + Code blocks
4. **Live Preview** - Real-time rendered preview
5. **Version Control** - Auto-save, version history, restore
6. **Typography Settings** - Font family, font size
7. **Export** - Markdown (PDF deferred)
8. **AI Chat** - NVIDIA API integration with streaming

// utils/jsonFile.ts

export async function readJSON<T>(filepath: string): Promise<T> {
  const data = await fs.readFile(filepath, 'utf-8');
  return JSON.parse(data) as T;
}

export async function writeJSON<T>(filepath: string, data: T): Promise<void> {
  const tempPath = filepath + '.tmp';
  await fs.writeFile(tempPath, JSON.stringify(data, null, 2), 'utf-8');
  await fs.rename(tempPath, filepath); // Atomic on most filesystems
}

export async function ensureDir(dirpath: string): Promise<void> {
  try {
    await fs.access(dirpath);
  } catch {
    await fs.mkdir(dirpath, { recursive: true });
  }
}

## 5. AI Integration

### 5.1 NVIDIA API Proxy

```typescript
// services/aiService.ts

interface AIRequest {
  messages: Array<{
    role: 'system' | 'user' | 'assistant';
    content: string;
  }>;
  stream: boolean;
  temperature: number;
  topP: number;
  maxTokens: number;
}

export async function streamAIChat(
  request: AIRequest,
  onChunk: (chunk: string) => void,
  onComplete: () => void,
  onError: (error: Error) => void
): Promise<() => void> {
  const controller = new AbortController();

  const response = await fetch('https://integrate.api.nvidia.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.NVIDIA_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      model: 'minimaxai/minimax-m2.7',
      messages: request.messages,
      stream: true,
      temperature: request.temperature,
      top_p: request.topP,
      max_tokens: request.maxTokens,
    }),
    signal: controller.signal,
  });

  const reader = response.body?.getReader();
  const decoder = new TextDecoder();

  async function read() {
    const { done, value } = await reader!.read();
    if (done) {
      onComplete();
      return;
    }
    const chunk = decoder.decode(value);
    // Parse SSE format: data: {...}
    const lines = chunk.split('\n');
    for (const line of lines) {
      if (line.startsWith('data: ')) {
        const data = JSON.parse(line.slice(6));
        if (data.choices?.[0]?.delta?.content) {
          onChunk(data.choices[0].delta.content);
        }
      }
    }
    read();
  }

  read();

  return () => controller.abort();
}
```

### 5.2 System Prompt Builder

```typescript
function buildSystemPrompt(cursorRules: CursorRule[], chapterContext: string): string {
  const enabledRules = cursorRules.filter((r) => r.enabled);

  let prompt = `You are an AI writing assistant for a book editor.
Help with writing, editing, formatting (Markdown, LaTeX), and following structure guidelines.
When providing suggestions, provide 2-3 options labeled A, B, C.`;

  if (enabledRules.length > 0) {
    prompt += `\n\nCURSOR RULES:\n`;
    enabledRules.forEach((rule, i) => {
      prompt += `${i + 1}. ${rule.content}\n`;
    });
  }

  if (chapterContext) {
    prompt += `\n\nCURRENT CHAPTER:\n${chapterContext}\n`;
  }

  return prompt;
}
```
