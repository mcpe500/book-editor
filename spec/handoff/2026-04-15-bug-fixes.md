## 4. Key Decisions Made

### 4.1 Markdown Pipeline
Removed custom `processTables` and `processCodeBlocks` pre-processing. `marked.parse()` handles these natively. Custom pre-processing was causing double-rendering issues.

### 4.2 Syntax Highlighting
Used `highlight.js` with custom marked renderer. Only registered 3 languages (typescript, python, javascript) to keep bundle small. Can add more as needed.

### 4.3 Dark Mode
Implemented via CSS custom properties (`--bg-primary`, etc.) with `[data-theme]` attribute selector. System theme uses `@media (prefers-color-scheme: dark)`.

### 4.4 Auto-Save
Renamed `handleAutoSave` to `scheduleAutoSave` to clarify it schedules rather than executes immediately. Added proper cleanup in `onMount` return.
